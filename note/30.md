# 运行调试写屏程序

查看内存可以使用调试命令xp

xp /512xb 0x7c00

   参数以/开始

   512kb

   512x x表示16进制显示

   512xb b表示以字节为单位

   512xb 表示以16进制显示数据，以字节为单位，显示512个字节

   最后给出物理地址 0x7c00 

```
xp /512
```
