## b 断点指令

```
b
```

实现设置一个内存地址，当处理器执行到这个地址时，自动停下来。

计算机启动后总是把主引导程序地址加载到物理地址0x7c00处，然后从这里取指令并执行，所以可以将这个地址设置为断点。

![image.png](https://cdn.nlark.com/yuque/0/2021/png/288075/1616574569758-d2635986-97a9-4272-a74b-ab6cb4b81752.png)

```
c
```
